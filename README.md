# Avoid

A minimal Linux distribution based on Void Linux, aimed at servers, build machines, and live recovery disks.

## Images

Pre-built images are available on the [releases](https://github.com/sku0x20/avoid/releases) page:

- `avoid.img.gz` — raw disk image, use with `dd` after unzipping; won't work with CDs. Should work with USBs and Ventoy.
- `avoid.qcow2` — Cloud image

No ISO for now.

## Build

Check packages.list for included packages.
It also includes minimal configs for some of the tools.
Check out build scripts for more info.
